# World Happiness Index Analysis

## Overview

This project analyzes the World Happiness Index dataset using Python and visualization techniques.

## Tools Used

- Python

- Pandas

- NumPy

- Matplotlib

- Jupyter Notebook

## Project Files

- happinessdataset.csv

- cleaned_world_happiness.csv

- world_happiness_analysis.ipynb

- app.py

## Key Insights

- GDP positively affects happiness score

- Social support impacts rankings

- Countries with lower corruption tend to rank higher
